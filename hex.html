Definição do Sistema Hexadecimal

O sistema de numeração hexadecimal ou também chamado de Base 16 possui dezesseis dígitos, 0, 1, 3, 4, 5, 6, 7, 8, 9, A, B, C, D, E, F.
, A, B, C, D, E, F. A forma de arranjar os 16 símbolos para representar as diferentes quantidades na Base 16 é a mesma usada anteriormente. Só que agora temos mais símbolos do que a Base Decimal. Quando os 16 símbolos são todos usados, os próximos devem ser arranjos do 1 com cada um dos 16 símbolos. conversão

Base 10 para base 16

1-Primeiro deve-se dividir o número decimal sucessivamente por dezesseis e guardar o resultado.

2-O processo deve ser repetido até que o quociente seja igual a 16.

3- A conversão será obtida com a sequência dos restos junto com o último quociente, começando pela direita.

Base 16 para base 10

1- Seguindo a mesma regra dos outros sistemas, deve-se decompor o número e multiplica-lo por uma potência de base 16.

base 2 para base 16

- Deve-se usar a tabela de equivalência, primeiramente separando o número binário em grupos de quatro dígitos, logo depois os substituindo por seus correspondentes na tabela de equivalência
